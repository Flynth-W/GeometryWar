= Geometry War

    a simple game, you just have to destroy squares

video::./playing.mp4[]

== Libraries

*   GLFW
*   OpenGl/glew
*   glm
*   irrKlang ( audio )

== Compile

supported for arch linux

    install packages:
        $ pacman -S glew glm glfw-x11 
    Compile with Makefile
        $ cd ./build/
        $ make
